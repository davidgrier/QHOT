SLM Tab

The SLM tab shows a live preview of the phase hologram currently being displayed on the spatial light modulator (SLM).

The preview updates automatically each time a new hologram is computed. While the tab is not visible the preview is paused, so switching away from it has no performance cost.

Navigating the preview

Zoom in / out
Scroll the mouse wheel over the preview image.
Zoom to region
Right-click and drag to draw a box around the area of interest.
Pan
Left-click and drag to scroll around the image.
Reset view
Right-click and choose View All from the context menu to return to the full hologram view.

The phase pattern is displayed as a grayscale image: pixel brightness represents phase from 0 (black) to 2π (white), encoded as an 8-bit value in the range 0–255.
