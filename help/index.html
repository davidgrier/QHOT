QHOT User Manual

QHOT is a point-and-click interface for holographic optical trapping and video microscopy. It uses a spatial light modulator (SLM) to project configurable optical traps into a microscope field of view.

QHOT's controls are grouped into tabs in the bar above this browser. Navigate onto a tab by clicking on it. This Help Browser is located in the Help tab. What the tabs control is described in the links below.
